# Code of Conduct

이 프로젝트는 질문하기 쉬운 분위기와, 초보자에게도 친절한 문서를 만드는 문화를 지향한다.

## 우리가 기대하는 태도

- 사람을 공격하지 않고 아이디어와 구현을 비판한다.
- 질문 수준으로 상대를 평가하지 않는다.
- 피드백은 구체적으로 하고, 비꼬는 말투는 피한다.
- 실수는 빠르게 인정하고 함께 고친다.

## 허용하지 않는 행동

- 개인 공격, 모욕, 차별, 괴롭힘
- 반복적인 비아냥, 조롱, 배제
- 사적인 정보의 무단 공개
- 협업을 어렵게 만드는 공격적 언행

## 운영 원칙

- 유지관리자는 상황에 따라 경고, 대화 중단, 기여 제한 같은 조치를 할 수 있다.
- 문제가 생기면 저장소 이슈나 PR에서 싸우기보다 유지관리자에게 조용히 알려 해결하는 쪽을 우선한다.

문서 품질만큼이나 협업 품질도 중요하다.
좋은 위키는 좋은 분위기에서 더 잘 나온다.
