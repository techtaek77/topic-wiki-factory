# chess-intro

체스를 처음 배우는 사람을 위한 knowledge 유형 샘플 위키다.
검증용 아티팩트 전체를 싣는 대신, 공개 저장소에서 보여주기 좋은 최소 예시만 담았다.

## What this example shows

- `wiki-config.yaml` - 어떤 입력으로 시작했는지
- `wiki-state.json` - 어떤 문서들이 planned/written/done 상태로 관리되는지
- `output/` - 실제 생성 결과가 어떤 모양으로 나오는지
- `output/assets/*.svg` - 말 배치나 이동처럼 글보다 그림이 빠른 부분을 어떻게 보여줄지

## Notes

- 실제 프로젝트에서는 `output_path`를 원하는 위치로 바꿔 쓸 수 있다.
- 이 예시는 `intro` 깊이의 작은 샘플이라 문서 수를 줄여 놨다.
- 링크 표기는 Obsidian/GitHub Markdown 둘 다 의식한 위키 스타일을 유지했다.
