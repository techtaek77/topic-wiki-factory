# 룩 (Rook)

> 가로와 세로 방향으로 원하는 만큼 이동할 수 있는 기물이다.

---

## 왜 필요한가

룩은 움직임이 단순해서 초보자가 이해하기 쉽고, 후반으로 갈수록 존재감이 커지는 기물이다.
오픈 파일을 잡으면 상대 진영을 멀리서 압박할 수 있어서 "기물을 어디에 두어야 힘이 생기는지"를 배우기 좋다.

- 없으면: 열린 줄을 장악하는 감각이 잘 안 생긴다
- 있으면: 단순한 직선 이동만으로도 강한 압박을 만들 수 있다
- 비유: 길만 열리면 끝까지 달리는 기차 같다

## 어떻게 적용하는가

초반에는 다른 기물과 폰이 많아서 룩이 바로 활약하기 어렵다.
그래서 룩은 서두르지 말고, 길이 열릴 때 중앙이나 열린 파일로 들어가는 식으로 다루면 된다.

### 예시

백의 e폰과 흑의 e폰이 교환돼 e파일이 비었다고 해보자.
이때 백 룩을 `e1`에 두면 세로줄 전체를 감시할 수 있고, 상대가 `e7`이나 `e8`에 기물을 둘 때 큰 압박이 생긴다.

### 핵심 포인트

- 룩은 열린 줄에서 가장 힘이 세다.
- 초반에 억지로 끌고 나오기보다 캐슬링 뒤 자연스럽게 중앙으로 가져오는 편이 좋다.
- 두 룩이 서로 이어지면 압박이 훨씬 강해진다.

### 자주 하는 실수

- 초반부터 룩만 앞세움 -> 다른 기물에 막혀 되돌아오기 쉽다.
- 폰 하나 잡겠다고 옆길로 샘 -> 룩의 큰 장점인 줄 장악을 놓친다.
