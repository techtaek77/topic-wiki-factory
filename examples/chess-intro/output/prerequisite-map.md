# 시작 전에 알아야 할 것

> 체스를 처음 볼 때는 "기물 이름 -> 체스판 좌표 -> 체크와 승패" 순서로 익히면 덜 헷갈린다.

## 최소 선수 지식

| 항목 | 왜 먼저 필요한가 | 다음 문서 |
|------|------------------|-----------|
| 기물 이름 | 룩, 나이트, 비숍을 구분해야 설명을 따라갈 수 있다 | [glossary](glossary.md) |
| 체스판 좌표 | 문서 속 `e4`, `h8` 같은 표기를 읽으려면 필요하다 | [quick-start](docs/guides/quick-start.md) |
| 체크 개념 | 왕을 지키는 목적을 알아야 수의 의미가 생긴다 | [faq](faq.md) |

## 가장 짧은 학습 경로

1. [glossary](glossary.md)로 기물 이름과 기본 용어를 익힌다.
2. [quick-start](docs/guides/quick-start.md)로 말 놓기와 첫 수 흐름을 따라간다.
3. [rook](docs/concepts/rook.md)을 읽으며 실제 기물 하나를 깊게 이해한다.

## 메모

- 처음부터 오프닝 이름을 외울 필요는 없다.
- 규칙을 다 외우는 것보다 한 판을 끝까지 따라가는 경험이 먼저다.
