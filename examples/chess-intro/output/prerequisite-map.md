# 시작 전에 알아야 할 것

> 체스를 처음 볼 때는 "기물 이름 -> 체스판 좌표 -> 체크와 승패" 순서로 익히면 덜 헷갈린다.

## 최소 선수 지식

| 항목 | 왜 먼저 필요한가 | 다음 문서 |
|------|------------------|-----------|
| 게임 목표 | 체크메이트가 무엇인지 알아야 왜 움직이는지 이해된다 | [chess-basics](docs/guides/chess-basics.md) |
| 기물 이름 | 킹, 퀸, 룩, 비숍, 나이트, 폰을 구분해야 설명을 따라갈 수 있다 | [glossary](glossary.md) |
| 체스판 좌표 | 문서 속 `e4`, `h8` 같은 표기를 읽으려면 필요하다 | [quick-start](docs/guides/quick-start.md) |
| 체크 개념 | 왕을 지키는 목적을 알아야 수의 의미가 생긴다 | [faq](faq.md) |

## 가장 짧은 학습 경로

1. [chess-basics](docs/guides/chess-basics.md)로 게임 목표와 기물 움직임을 잡는다.
2. [glossary](glossary.md)로 용어를 빠르게 훑는다.
3. [quick-start](docs/guides/quick-start.md)로 말 놓기와 첫 수 흐름을 따라간다.
4. [rook](docs/concepts/rook.md)을 읽으며 실제 기물 하나를 깊게 이해한다.
5. [sources](sources.md)에서 사이트나 유튜브 하나를 골라 반복 연습한다.

## 하루 학습 루트 예시

| 시간 | 할 일 | 목표 |
|------|-------|------|
| 10분 | [chess-basics](docs/guides/chess-basics.md) 읽기 | 말이 왜 움직이는지 큰 그림 이해 |
| 10분 | [quick-start](docs/guides/quick-start.md) 따라 배치 | 실제 판 감각 만들기 |
| 10분 | [faq](faq.md) 훑기 | 초반 헷갈림 제거 |
| 20분 | [sources](sources.md)에서 사이트 한 곳으로 연습 | 읽은 내용을 실제 수로 연결 |

## 메모

- 처음부터 오프닝 이름을 외울 필요는 없다.
- 규칙을 다 외우는 것보다 한 판을 끝까지 따라가는 경험이 먼저다.
- 유튜브는 좋지만, 말이 어디로 가는지 직접 움직여 보지 않으면 기억이 훅 날아간다.
