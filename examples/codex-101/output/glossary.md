# 용어집

## Agent

정해진 역할과 지시를 기준으로 작업을 수행하는 AI 협업 단위다.

## Context

현재 작업을 이해하는 데 필요한 배경, 제약, 파일, 목적을 묶어 부르는 말이다.

## Review

버그, 회귀, 빠진 테스트, 위험한 가정을 찾는 검토 단계다.

## Implementation

실제로 코드를 쓰거나 파일을 고치는 생산 단계다.

## Feature Branch

작업 단위를 메인 브랜치와 분리해 안전하게 진행하는 브랜치다.

## Harness

약속한 규칙이 깨지지 않았는지 검증하는 테스트용 점검 장치다.
