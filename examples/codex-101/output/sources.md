## Topic interpretation draft

- `Codex 101`은 제품 최신 기능 목록이 아니라, 코딩 에이전트와 함께 일하는 기본 협업 흐름을 설명하는 입문 위키로 해석한다.
- 범위는 task 분해, context 전달, review 루프, feature branch 같은 실무 기본기로 제한한다.
- 계정 정책, 가격, 자주 바뀌는 세부 UI는 범위 밖으로 둔다.
