# 먼저 알면 덜 헷갈리는 것

## 바로 시작해도 되는 상태

- 자동화가 항상 좋은 건 아니라는 감각이 있다.
- 승인, 검토, 리뷰가 왜 필요한지 대충은 안다.
- 실패 비용이 큰 작업과 작은 작업이 다르다는 걸 이해한다.

## 여기서 자주 막히는 지점

- 승인 단계를 넣으면 다 안전해진다고 생각하기 쉽다.
- 사람 검토자에게 필요한 맥락을 안 주고 버튼만 달아 두기 쉽다.
- 중요하지 않은 단계까지 전부 수동 검토로 묶어 속도를 죽이기 쉽다.
