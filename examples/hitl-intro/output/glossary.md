# 용어집

- **HITL**: 자동화 흐름 안에 사람 검토나 승인 단계를 넣는 방식
- **checkpoint**: 사람이 개입하도록 설계한 특정 단계
- **approval**: 여러 선택지 중 하나를 사람이 승인하는 행위
- **fallback**: 자동 판단이 애매할 때 사람이 넘겨받는 우회 경로
- **escalation**: 위험도나 불확실성이 커질 때 더 강한 사람 검토로 넘기는 것
- **false positive**: 문제 없는데 문제라고 잘못 잡는 경우
- **false negative**: 문제 있는데 놓치는 경우
